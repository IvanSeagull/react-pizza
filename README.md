# 🍕 Pizza Shop

## A small pet project in which I made a prototype of a pizza store with a menu and a basket

# 🛠 Stack:

- **ReactJS 18**
- **TypeScript**
- **Redux Toolkit**
- **React Router v6**
- CSS-Modules / SCSS
- React Content Loader (Skeleton)
- React Pagination
- Lodash.Debounce
- Code Splitting, React Loadable
